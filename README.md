# 🏰 Medieval 3D Scene - OpenGL

[cite_start]Proiect de Prelucrare Grafică realizat de **Petean Irina**. [cite: 3]
[cite_start]O scenă complexă medieval-fantasy ce include animații, iluminare Phong și umbre dinamice. [cite: 2, 22, 209]

## 📥 Descărcare Resurse (Modele 3D)
**Atenție:** Din cauza limitelor de mărime ale GitHub, folderul `models/` nu este inclus în acest repozitoriu.
1. Descărcați modelele și texturile de aici: **[https://drive.google.com/file/d/1FvTU4GDQ-yWYy9gA7-bK8EKBvmX2T9sF/view?usp=sharing]**
2. Extrageți arhiva și plasați folderul `models/` în rădăcina proiectului înainte de rulare.

## ✨ Funcționalități Tehnice
* [cite_start]**Sistem de Iluminare**: Model Phong cu lumină direcțională (soare) și punctiformă (felinar). [cite: 205, 206, 210]
* [cite_start]**Umbre Dinamice**: Tehnica Shadow Mapping implementată prin Framebuffer Objects (FBO). [cite: 283, 301, 302]
* [cite_start]**Efecte Atmosferice**: Ceață exponențială și sistem de particule pentru ploaie. [cite: 364, 366, 384, 385]
* [cite_start]**Animații**: Mișcări bazate pe timp pentru dragon și vultur. [cite: 405, 406, 429]
