# 🏰 Medieval 3D Scene - OpenGL

Proiect de Prelucrare Grafică realizat de **Petean Irina**. 
O scenă complexă medieval-fantasy ce include animații, iluminare Phong și umbre dinamice.

## 📥 Descărcare Resurse (Modele 3D)
**Atenție:** Din cauza limitelor de mărime ale GitHub, folderul `models/` nu este inclus în acest repozitoriu.
1. Descărcați modelele și texturile de aici: **https://drive.google.com/file/d/1FvTU4GDQ-yWYy9gA7-bK8EKBvmX2T9sF/view?usp=sharing**
2. Extrageți arhiva și plasați folderul `models/` în rădăcina proiectului înainte de rulare.

## ✨ Funcționalități Tehnice
* **Sistem de Iluminare**: Model Phong cu lumină direcțională (soare) și punctiformă (felinar). 
* **Umbre Dinamice**: Tehnica Shadow Mapping implementată prin Framebuffer Objects (FBO). 
* **Efecte Atmosferice**: Ceață exponențială și sistem de particule pentru ploaie. 
* **Animații**: Mișcări bazate pe timp pentru dragon și vultur. 

## 🛠️ Tehnologii
* **Limbaj**: C++ 
* **API Grafic**: OpenGL Core Profile 
* **Biblioteci**: GLFW, GLEW, GLM
